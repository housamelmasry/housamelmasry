## 👨‍💻 About Me

> Backend Developer with **4+ years** building production systems — from ERP/CRM platforms to a Quran memorization app with pixel-accurate Mushaf rendering.

I specialize in **Laravel & PHP** backend architecture, and I've shipped complete systems independently — database schema, REST APIs, admin panels, and mobile apps — all into real production environments.

What sets me apart:

- 📖 **Built a Duolingo-style Quran memorization app** with authentic Arabic glyph rendering that matches real Mushaf page layouts — a technically complex problem few developers have tackled
- 🎙️ **Currently building a Tajweed voice comparison engine** — on-device algorithm that compares user recitation against reference recordings word-by-word to detect Madd, Sakta, and other Tajweed rules
- 🏗️ **4+ years in Laravel** — ERP/CRM systems, multi-branch POS, event-driven billing, multi-role RBAC
- 📱 **React Native CLI** with custom native modules for iOS (Swift) & Android (Kotlin)
- 🎯 **Expanding into Next.js** to bridge backend expertise with modern full-stack development
