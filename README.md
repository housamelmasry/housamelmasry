### 🍞 Sourdough — Complete Multi-Platform Ecosystem

> A fully live production system serving real customers in Alexandria, Egypt.
> I designed and built the **entire stack solo**.

| Component | Description | Link |
|-----------|-------------|------|
| **CMS** | Laravel + Filament — 3 panels: Admin, Delivery, Branch | [sour-cms.online](https://sour-cms.online) |
| **Mobile App** | React Native CLI — published on Google Play | [Google Play](https://play.google.com/store/apps/details?id=com.sourdough.sourdoughapp) |
| **POS Tablet App** | React Native CLI — cashier & waiter roles, PIN auth, full order lifecycle | APK on request |
| **E-commerce Site** | WordPress / WooCommerce — full catalog & online ordering | [sourdough.house](https://www.sourdough.house) |

**Stack:** `Laravel` `Filament` `MySQL` `REST API` `Spatie` `React Native CLI` `WordPress`

**What I built:**
- Multi-branch inventory management with recipe-based cost price engine
- POS system: dine-in / takeaway / delivery order lifecycle → kitchen → close → invoice
- REST API with JWT auth + RBAC across admin / branch manager / staff / cashier roles
- Event-driven billing: `AppointmentCompleted` / `OrderCompleted` listeners auto-generate invoices
- Android mobile app synced in real-time with the CMS
